## Setup

```bash
  - node -v

  - npm init -y

  - npm install -D tailwindcss

  - npx tailwindcss init

  # run
  - npm run watch
```

## and:

In package.json you change:

- "scripts": {
  "watch": "tailwindcss -i ./input.css -o ./output.css --watch";
  }
- In text editor you install:

  - Live Server
  - Tailwind CSS IntelliSense
